Il est vrai que l\u2019eau est un \u00e9l\u00e9ment indispensable pour toute vie sur terre. Et le mot  “eau” – un nom f\u00e9minin est \u00e9galement employ\u00e9 dans plusieurs des expressions de tous les jours dans la langue fran\u00e7aise. <\/p>\n\n\n\n

Alors, d\u00e9couvrons les 15 expressions courantes sur l\u2019eau dans cet \u00e9pisode !\ud83d\udca6<\/p>\n\n\n\n

1- Comme un poisson dans l’eau <\/strong><\/p>\n\n\n\n

\u2714Signification : \u00catre tr\u00e8s \u00e0 l\u2019aise<\/p>\n\n\n\n

\u2714Exemple : Je me sens comme un poisson dans l’eau dans ma nouvelle \u00e9quipe.<\/p>\n\n\n\n

2- Clair comme de l’eau de roche<\/strong><\/p>\n\n\n\n

\u2714Signification : \u00catre \u00e9videntes<\/p>\n\n\n\n

\u2714Exemple : Ton explication est parfaite. J’ai bien compris, c’est clair comme de l’eau de roche.<\/p>\n\n\n\n

3- \u00c0 l\u2019eau de rose <\/strong><\/p>\n\n\n\n

\u2714\ufe0f<\/strong>Signification : Sentimental, jouant sur des \u00e9motions simples, manque de vigueur<\/p>\n\n\n\n

\u2714Exemple : C’est un roman \u00e0 l’eau de rose.<\/p>\n\n\n\n

3- Tomber \u00e0 l\u2019eau<\/strong><\/p>\n\n\n\n

\u2714Signification : \u00catre annul\u00e9 ou abandonn\u00e9, ne plus \u00eatre envisag\u00e9<\/p>\n\n\n\n

\u2714Synonyme : Tomber dans le lac, tomber dans le panneau<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : L’\u00e9v\u00e9nement est annul\u00e9. La soir\u00e9e qu’on avait pr\u00e9vue tombe \u00e0 l’eau.<\/p>\n\n\n\n

4- \u00catre sous l\u2019eau <\/strong><\/p>\n\n\n\n

\u2714Signification : \u00catre d\u00e9bord\u00e9, souvent par une quantit\u00e9 de travail ou de t\u00e2ches \u00e0 r\u00e9aliser<\/p>\n\n\n\n

\u2714Synonyme : \u00catre submerg\u00e9<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : Je suis sous l\u2019eau et n’arrive pas \u00e0 faire tout mon travail.<\/p>\n\n\n\n

5- Se jeter \u00e0 l\u2019eau <\/strong><\/p>\n\n\n\n

\u2714\ufe0fSignification : Se forcer \u00e0 prendre une d\u00e9cision importante et \u00e0 prendre des risques<\/p>\n\n\n\n

\u2714Exemple : C’est d\u00e9cid\u00e9, je me jette \u00e0 l’eau : demain je lui avoue tout !<\/p>\n\n\n\n

6- Aller \u00e0 vau-l\u2019eau <\/strong><\/p>\n\n\n\n

\u2714\ufe0f<\/strong>Signification : Courir \u00e0 sa perte, aller sans contr\u00f4le<\/p>\n\n\n\n

\u2714Exemple : Vous allez \u00e0 vau-l’eau en n’\u00e9coutant pas mes conseils.<\/p>\n\n\n\n

7- Avoir l\u2019eau \u00e0 la bouche <\/strong><\/p>\n\n\n\n

\u2714Signification : Saliver d\u2019envie, \u00eatre attir\u00e9 par quelque chose qui fait envie ou curiosit\u00e9 <\/p>\n\n\n\n

\u2714\ufe0fSynonyme : Mettre<\/em><\/strong> <\/em>l\u2019eau \u00e0 la bouche<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : Rien que de lire le menu, \u00e7a met d\u00e9j\u00e0 l\u2019eau \u00e0 la bouche !<\/p>\n\n\n\n

8- Mettre de l\u2019eau dans son vin <\/strong><\/p>\n\n\n\n

\u2714Signification : \u00catre plus mod\u00e9r\u00e9, mod\u00e9rer ses exigences<\/p>\n\n\n\n

\u2714Exemple : Briel est plus agr\u00e9able depuis qu’il a mis de l’eau dans son vin.<\/p>\n\n\n\n

9- Se m\u00e9fier de l\u2019eau qui dort<\/strong><\/p>\n\n\n\n

\u2714Signification : Se m\u00e9fier des individus calmes car il peuvent s\u2019av\u00e9rer \u00eatre les plus dangereux<\/p>\n\n\n\n

\u2714Exemple : On dit qu’il faut se m\u00e9fier de l’eau qui dort.<\/p>\n\n\n\n

10- Jeter le b\u00e9b\u00e9 avec l’eau du bain<\/strong><\/p>\n\n\n\n

\u2714Signification : Perdre de vue l’essentiel, ne pas prendre la peine de s\u00e9parer le bon du mauvais et tout jeter<\/p>\n\n\n\n

\u2714Exemple : Cependant, il jette le b\u00e9b\u00e9 avec l’eau du bain, et cette attitude est irresponsable.<\/p>\n\n\n\n

11- Se ressembler comme deux gouttes d\u2019eau<\/strong><\/p>\n\n\n\n

\u2714Signification : Se ressembler de mani\u00e8re parfaitement identique<\/p>\n\n\n\n

\u2714Exemple : Elles se ressemblent comme deux gouttes d’eau.<\/p>\n\n\n\n

12- Il y a de l’eau dans le gaz.<\/strong><\/p>\n\n\n\n

\u2714\ufe0fSignification : \u00c7a ne va pas tr\u00e8s bien et l’atmosph\u00e8re est tendue.<\/p>\n\n\n\n

\u2714\ufe0fExemple : J’entends souvent les voisins se disputer. Il y a de l’eau dans le gaz dans leur couple.<\/p>\n\n\n\n

13- De l’eau a coul\u00e9 sous les ponts.<\/strong><\/p>\n\n\n\n

\u2714Signification : Le temps a pass\u00e9, les choses ont \u00e9volu\u00e9 et ne ressemblent plus \u00e0 ce qu\u2019elles \u00e9taient avant.<\/p>\n\n\n\n

\u2714\ufe0fSynonyme : De l’eau est pass\u00e9e sous les ponts.<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : L’eau a coul\u00e9 sous les ponts depuis la derni\u00e8re fois que nous nous sommes rencontr\u00e9s.  <\/p>\n\n\n\n

14- C\u2019est une goutte d\u2019eau dans la mer.<\/strong><\/p>\n\n\n\n

 <\/strong>\u2714\ufe0fSignification :<\/strong> C\u2019est une effet n\u00e9gligeable, insignifiant, sans cons\u00e9quence.<\/p>\n\n\n\n

\u2714\ufe0fSynonyme : C\u2019est une goutte d\u2019eau dans l\u2019oc\u00e9an.<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : Le gouvernement a pris cette mesure mais c\u2019est une goutte d\u2019eau dans la mer. \u00c7a ne va pas r\u00e9soudre le probl\u00e8me.<\/p>\n\n\n\n

15- C’est la goutte d’eau qui fait d\u00e9border le vase. <\/strong><\/p>\n\n\n\n

\u2714Signification : C\u2019est la petite chose, le petit probl\u00e8me qui vient se rajouter \u00e0 une situation d\u00e9j\u00e0 difficile.<\/p>\n\n\n\n

\u2714Synonyme : La coupe est pleine.<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : J’en ai assez ! L\u00e0, c’est la goutte d’eau qui fait d\u00e9border le vase !<\/p>\n\n\n\n
