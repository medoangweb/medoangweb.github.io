Le cheval est un grand mammif\u00e8re herbivore et ongul\u00e9 \u00e0 sabot unique. #1 – Un rem\u00e8de de cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Un rem\u00e8de particuli\u00e8rement fort et violent, mais qui a vocation \u00e0 \u00eatre efficace<\/p>\n\n\n\n

\u2714\ufe0fSynonyme : Une m\u00e9decine de cheval<\/em><\/strong><\/p>\n\n\n\n

\u2714\ufe0fExemple : \u00c9coutez, donnez-moi un rem\u00e8de de cheval contre la douleur.<\/p>\n\n\n\n

#2 – Une sant\u00e9 de cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Une excellente sant\u00e9<\/p>\n\n\n\n

\u2714Exemple : Il faut avoir une sant\u00e9 de cheval pour \u00e9tudier et travailler efficacement.<\/p>\n\n\n\n

#3 – Un travail de cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Une t\u00e2che tr\u00e8s p\u00e9nible<\/p>\n\n\n\n

\u2714Exemple : C\u2019est un travail de cheval.<\/p>\n\n\n\n

#4 – Un grand cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Une grande femme aux allures masculines<\/p>\n\n\n\n

\u2714Exemple : Ils pensent qu\u2019elle est un grand cheval.<\/p>\n\n\n\n

#5 – Un cheval de troie <\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Un cadeau mal\u00e9fique<\/p>\n\n\n\n

\u2714Exemple : Elle m\u2019a donn\u00e9 un cadeau mal\u00e9fique.<\/p>\n\n\n\n

#6 – Un cheval de retour <\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Un r\u00e9cidiviste<\/p>\n\n\n\n

\u2714Exemple : Il est vraiment un cheval de retour.<\/p>\n\n\n\n

#7 – \u00catre \u00e0 cheval sur <\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : \u00catre tr\u00e8s exigeant, \u00eatre tr\u00e8s strict sur, attacher une grande importance<\/p>\n\n\n\n

\u2714\ufe0fExemple : Elle est \u00e0 cheval sur la  politesse. <\/p>\n\n\n\n

#8 – Ne pas \u00eatre un mauvais cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : \u00catre <\/strong>gentil, \u00eatre de bonne volont\u00e9, ne pas \u00eatre si mauvais<\/p>\n\n\n\n

\u2714\ufe0fExemple : Il est vrai qu\u2019il n\u2019est pas un mauvais cheval.<\/p>\n\n\n\n

#9 – Avoir mang\u00e9 du cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Avoir une \u00e9nergie inhabituelle, faire preuve d’une \u00e9nergie inaccoutum\u00e9e<\/p>\n\n\n\n

\u2714\ufe0fExemple : Pourquoi t\u2019as mang\u00e9 du cheval ?<\/p>\n\n\n\n

#10 – Avoir une fi\u00e8vre de cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Avoir une fi\u00e8vre tr\u00e8s \u00e9lev\u00e9e, avoir une grosse fi\u00e8vre, avoir une fi\u00e8vre carabin\u00e9e<\/p>\n\n\n\n

\ud83d\udc49Cette expression famili\u00e8re indique qu’une personne a une temp\u00e9rature tr\u00e8s \u00e9lev\u00e9e, qui avoisine souvent les 40\u00b0.<\/em><\/p>\n\n\n\n

\u2714\ufe0fExemple :  Je suis arriv\u00e9 l\u00e0-bas avec une fi\u00e8vre de cheval.<\/p>\n\n\n\n

#11 – Avoir un cheval de bataille<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Avoir un argument favori et coutumier, avoir un sujet favori pour d\u00e9fendre une cause, un th\u00e8me de pr\u00e9dilection<\/p>\n\n\n\n

\u2714\ufe0fExemple : Ce fut le cheval de bataille de la commission des budgets.<\/p>\n\n\n\n

#12 –  Manger avec les chevaux de bois <\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Ne rien avoir \u00e0 manger, je\u00fbner<\/p>\n\n\n\n

\u2714Exemple : Le frigo est vide, je dois manger avec les chevaux de bois. <\/p>\n\n\n\n

#13 – L\u00e2cher les chevaux<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Tout donner, aller jusqu’au bout<\/p>\n\n\n\n

\u2714\ufe0fExemple : Il faut l\u00e2cher les chevaux pour atteindre nos objectifs plus \u00e9lev\u00e9s. <\/p>\n\n\n\n

#14 – Changer un cheval borgne pour un aveugle<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Faire un march\u00e9 de dupes, \u00e9carter sa propre solution, quoiqu’ imparfaite, pour une autre solution certainement plus catastrophique<\/p>\n\n\n\n

\u2714\ufe0fSynonyme : Troquer son cheval borgne contre un aveugle<\/em><\/strong><\/p>\n\n\n\n

\u2714\ufe0fExemple : De toute fa\u00e7on, pour elle, c’\u00e9tait changer son cheval borgne pour un aveugle.<\/p>\n\n\n\n

#15 – Miser sur le bon cheval<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Faire le bon choix<\/p>\n\n\n\n

\u2714\ufe0fAntonyme : Miser sur le mauvais cheval <\/em><\/strong>(Ne pas faire le bon choix, parier sur quelqu’un qui vous conduit \u00e0 l’\u00e9chec)<\/p>\n\n\n\n

\u2714\ufe0fExemple : Mon petit, tu as mis\u00e9 sur le bon cheval.<\/p>\n\n\n\n

#16 – Monter sur ses grands chevaux<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : S\u2019emporter tr\u00e8s vite, se mettre en col\u00e8re de fa\u00e7on orageuse<\/p>\n\n\n\n

\u2714\ufe0fExemple : Mon cousin monte sur vos grands chevaux pour de petites choses.<\/p>\n\n\n\n

#17 – Fermer l’\u00e9curie une fois les chevaux enfuis <\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : R\u00e9agir trop tard, attendre que quelque chose arrive pour s’en pr\u00e9munir<\/p>\n\n\n\n

\u2714Exemple : Il ne faut pas fermer l’\u00e9curie une fois les chevaux enfuis.<\/p>\n\n\n\n

#18 – C’est la mort du petit cheval.<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : C’est la fin. C’est tr\u00e8s grave.<\/p>\n\n\n\n

\u2714Exemple : Sinon, le reste de l’ann\u00e9e, c’est la mort du petit cheval pour tout le monde.<\/p>\n\n\n\n

#19 – Cela ne se trouve pas sous le pas d’un cheval.<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Il est difficile de se le procurer. C’est rare, tr\u00e8s difficile \u00e0 trouver.<\/p>\n\n\n\n

\u2714\ufe0fSynonyme : Cela ne se trouve pas sous les sabots d’un cheval. \u00c7a ne se trouve pas sous le pied d’un cheval.<\/em><\/strong><\/p>\n\n\n\n

\u2714Exemple : Selon la directrice, \u00e7a ne se trouve pas sous le pas d’un cheval.<\/p>\n\n\n\n

#20 – \u00c0 cheval donn\u00e9, on ne regarde pas les dents.<\/strong><\/p>\n\n\n\n

\u2714\ufe0fD\u00e9finition : Il faut toujours \u00eatre content d’un cadeau re\u00e7u, on ne doit pas critiquer un cadeau quand bien m\u00eame aurait-il un d\u00e9faut.<\/p>\n\n\n\n

\u2714\ufe0fSynonyme : \u00c0 cheval donn\u00e9, on ne regarde pas la bouche. \u00c0 cheval donn\u00e9, on ne regarde pas la bride.<\/em><\/strong><\/p>\n\n\n\n
